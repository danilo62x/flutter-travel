# Flutter Travel

> Viagens e reservas em Flutter

[![Licença: MIT](https://img.shields.io/badge/Licen%C3%A7a-MIT-blue.svg)](./LICENSE) ![Grátis](https://img.shields.io/badge/pre%C3%A7o-Gr%C3%A1tis-brightgreen)

App de viagens/reservas em Flutter 3.44 + Material 3, com 8 telas e tema claro/escuro: explorar destinos e experiências; detalhe do destino; hotéis com filtro de preço/estrelas; página do hotel com comodidades e mapa desenhado; reserva com datas e hóspedes; voucher com QR Code; minhas viagens; e perfil com fidelidade. Navegação por 4 abas + rotas go_router, arquitetura em camadas com MVVM, CustomPaint.

## 🧱 Stack

- Flutter 3.44
- Dart 3
- Material 3
- go_router
- provider
- http

## ✨ Recursos

- Flutter 3.44 (último stable) via FVM
- Material 3 com tema claro e escuro
- Arquitetura em camadas (data/domain/ui) + MVVM
- Navegação declarativa com go_router
- Modelos com JSON (fromJson/toJson) + serviço http
- Destinos populares, experiências e preços
- Layout responsivo (mobile, tablet, desktop)
- Android · iOS · Web · Windows + testes de widget

## 🚀 Como rodar

Requer o [Flutter SDK](https://docs.flutter.dev/get-started/install) instalado (canal stable).

```bash
flutter pub get
flutter run
```

Para rodar na web: `flutter run -d chrome`. Para gerar o APK: `flutter build apk`.
